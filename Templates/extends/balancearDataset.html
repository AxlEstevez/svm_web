Balancear y guardar dataset

Una vez que el dataset este limpio, se balancea con el fin de que no haya mas datos de una u otra categoria en el entrenamiento y esto pueda influir al momento de clasificar una nueva noticia. Para balancear nuestro dataset se utilizo la libreria imlearn y para guardarlo, recordando que el dataset original esta en formato json, se hizo uso del método to_json de la libreria pandas.
